## Crear entorno virtual

``` python3 -m venv venv ```
 
## Activar entorno virtual

``` source venv/bin/activate ```

## Instalar flask

```pip install flask```

## Instalar sqlalchemy

```pip install Flask-SQLAlchemy ```

## Instalar flask wtf (crear formularios rapidamente)

```pip install flask_wtf ```
